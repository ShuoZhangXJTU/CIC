# CIC

This repository contains the Pytorch implementation of our paper
[Learning to Check Contract Inconsistencies](https://ojs.aaai.org/index.php/AAAI/article/view/17698) (AAAI 2021).

## Usage
### Run

```bash
python main.py
```
